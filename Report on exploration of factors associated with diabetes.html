Project overview

The project explores the socio-demographic, biological, behavioral characteristics related to diabetes based on the responses collected among the adults in the United States using Behavioral Risk Factor Surveillance System (BRFSS) in 2023. Data Source: https://www.cdc.gov/brfss/annual_data/2023/files/LLCP2023XPT.zip

Categorical variables had factor levels like missing, don’t know/not sure, refused. Function was applied to replace NA on those for easier missing value calculation.

Code 
# List of variables with empty levels to be replaced with NA
to_be_replaced <- list(
  income_level = c("Don't know/Not sure/Missing"),
  physical_activity = c("Don't know/Not Sure","Refused"),
  smoking         = c("Don't know/Refused/Missing"),
  tobacco_use     = c("Don't know/Not Sure", "Refused"),
  alc_drnk_30days = c("Don't know/Not Sure","Refused"),
  diabetes_status = c("Don't know/Not Sure", "Refused")
  )

# Function to replace  
replace_with_na <- function(df, to_be_replaced) {
  for (v in names(to_be_replaced)) 
    {
    df[[v]][df[[v]] %in% to_be_replaced[[v]]] <- NA
    }
  return(df)
  }

# Applying function
short_brfss <- replace_with_na(short_brfss, to_be_replaced)

“Complete Case Analysis”, involving removing entire rows(cases) that contain any missing values would not be appropriate as this would reduce the sample size and also it would introduce bias as these data are not missing completely at random. Therefore, we would do “Available Case Analysis” where we use the data that are present for a respondent and skips for data that are missing. Example: For specific visualizations requiring selected variables, rows with missing values for all selected variables were dropped there to ensure comparative presentation. (Ref: https://www.tandfonline.com/doi/full/10.1080/10696679.2024.2376052#d1e575)

Descriptive statistics

1. Socio-demographic characteristics:

1.1 Population pyramid (Age & Sex):

A population pyramid is selected, because, although age was recorded as a numeric variable, individuals above 80 years were reported simply as “>80 years.” Calculating and reporting the mean or median age would not be representative of the study population.

Code

                labels = c("18-24", "25-29",

                           "40-44", "45-49",
                           "50-54", "55-59", 
                           "60-64", "65-69", 
                           "70-74", "75-79",

py_df <- as.data.frame(table(df$age_g, 
                             df$sex))
colnames(py_df) <- c("age", 
                     "sex", 
                     "count")

# Count observation for n
total_pop <- sum(py_df$Freq)

# Convert count to percentage
